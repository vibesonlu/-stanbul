# 🏢 BrokerFlow // Mobile Real Estate Deal Pipeline

A lightweight, responsive single-page application (SPA) dashboard designed for real estate agencies and independent brokers to track inventory, calculate potential commissions, and manage sales pipelines on the go. 

This entire project was engineered and deployed completely on a mobile device utilizing **github.dev** and **GitHub Pages**.

## ✨ Features

* **Dynamic Deal Pipeline:** A responsive, swipe-friendly Kanban board tracking properties through three critical lifecycle stages: *Active Listings*, *Offer Received*, and *Under Contract*.
* **Real-Time Financial Analytics:** Automatically aggregates total pipeline valuation and instantly computes estimated commission cuts (fixed at a standard 3% broker fee).
* **Instant Inventory Mutations:** Seamlessly add new listings with customizable details (Address, Asking Price, Property Layout Type) or clear closed deals natively from the UI.
* **Mobile-First UX:** Built using a dark mode theme with zero heavy external dependencies, utilizing Tailwind CSS via CDN for rapid asset delivery and performance on cellular networks.

---

## 🛠️ Tech Stack & Architecture

* **Frontend Structure:** HTML5 (Semantic Layouts)
* **Styling Framework:** Tailwind CSS v4 (Mobile Fluid Flexbox Design)
* **State Management & Logic:** Vanilla JavaScript (In-Memory Array Mutations & DOM Re-rendering Engines)
* **Hosting Platform:** GitHub Pages CDN

---

## 📱 Mobile Development Workflow

This project serves as a proof-of-concept demonstrating that complex web tooling can be built entirely without a desktop workstation:
1. **IDE:** Sandboxed VS Code ecosystem via `github.dev` in mobile Chrome (iOS).
2. **Layout Tweak:** Custom UI optimization (word wrap toggles, sidebar collapses) to handle code structures on compact viewports.
3. **Deployment:** Continuous delivery handled automatically via GitHub Pages build actions upon master-branch pushes.
